## Traffic Volume Prediction 
Pattern Analysis using Linear Algebra

### Problem Statement
Traffic congestion is a major issue in urban areas, influenced by multiple factors such as weather conditions, time, and environmental variables. Understanding and predicting traffic patterns can help in better planning and congestion management.
In this project, we apply concepts from linear algebra to analyze a real-world traffic dataset. The goal is to transform raw data into a structured mathematical form, reduce unnecessary information, and use mathematical techniques to predict traffic volume and identify patterns.

### Link to Dataset 
https://www.kaggle.com/datasets/anshtanwar/metro-interstate-traffic-volume

The dataset used is the Metro Interstate Traffic Volume dataset from Kaggle.
### Features include:
- Temperature
- Rainfall (rain_1h)
- Snowfall (snow_1h)
- Cloud coverage (clouds_all)
- Traffic volume (target variable)

Each row represents traffic conditions at a specific time.
